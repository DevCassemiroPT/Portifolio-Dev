Sobre

Olá! Eu sou o Igor Cassemiro, um apaixonado por tecnologia e pelo seu potencial de transformar o mundo. Em um ano de aprendizado autodidata, eu me dediquei a explorar e dominar diversas tecnologias, como:

HTML

Utilizo HTML para estruturar o conteúdo das páginas web, garantindo uma base sólida para o desenvolvimento.

CSS

Minhas habilidades em CSS me permitem estilizar e tornar as páginas visualmente atraentes. Trabalho com seletores, propriedades e animações para criar layouts responsivos e agradáveis aos olhos.

React

Tenho experiência com o framework JavaScript React. Com ele, sou capaz de criar interfaces de usuário eficientes e reutilizáveis. Componentização, gerenciamento de estado e roteamento são algumas das minhas habilidades nessa área.

JavaScript

Sou proficiente em JavaScript e o utilizo para adicionar interatividade e dinamismo aos sites. Posso criar funcionalidades como formulários interativos, animações e manipulação do DOM.

Git e GitHub

São ferramentas essenciais para controle de versão e colaboração em projetos. Utilizo o Git para versionar meu código e o GitHub para compartilhar e colaborar com outros desenvolvedores.

MySQL

Trabalhei com o sistema de gerenciamento de banco de dados MySQL. Isso inclui a criação de tabelas, consultas SQL e otimização de consultas para recuperação eficiente de informações.

Bootstrap

O Bootstrap pode ser usado no projeto de diferentes formas, como via CDN ou instalado via gerenciador de pacotes. Ele também usa Sass para uma arquitetura modular e customizável. O Bootstrap é uma ferramenta poderosa e versátil para o desenvolvimento web.

Como posso agregar no time ?

O meu objetivo é criar páginas web que sejam responsivas, semânticas e otimizadas, que proporcionem uma experiência incrível e um desempenho excelente aos usuários.
Estou sempre em busca de novos desafios e oportunidades de aprendizado. Quero fazer parte de uma equipe que seja criativa, colaborativa e comprometida com a qualidade e a inovação!
